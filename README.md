# Clasificación de Prendas con Inteligencia Artificial

## Descripción

Proyecto de Visión por Computadora para clasificar imágenes de prendas en tres categorías:

- Playera
- Pantalón
- Sudadera

Se evaluaron distintos modelos de Deep Learning.
